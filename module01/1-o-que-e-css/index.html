O que é CSS?

O CSS (folha de estilo em cascata) tem como objetivo estilizar um documento, o HTML, para que ele seja personalizado da maneira que o desenvolvedor preferir.

Sintaxe do CSS

Para utilizar o css, nós temos a seguinte estrutura:

  • seletor: que é o elemento que estamos chamando para ser estilizado;
  • as chaves: usada para agrupar as declarações (que pode ser uma ou mais);
  • propriedade e valores: onde indicaremos o que e como vamos estilizar tal elemento.
