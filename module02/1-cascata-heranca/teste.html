Cascata e Herança

Cascata

Aplicando o conceito de cascata, quando é aplicado mais de um seletor com a mesma especificidade, é aplicado o último escrito, o conceito de cascata.

Especificidade

Quando são aplicados seletores diferentes para o mesmo elemento, é aplicado o seletor que tem uma pontuação mais alta. Nesse caso, o STYLE INLINE é mais alto que a ID que é mais alta que a CLASSE que é mais alta que o NOME DO ELEMENTO, quanto mais genérico, mais baixa a pontuação.
